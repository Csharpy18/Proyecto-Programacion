PROJECT TITLE: Elecciones presidenciales en Costa Rica 2026.

PURPOSE OF PROJECT: Sistema de registro par las elecciones presidenciales en Costa Rica.

VERSION or DATE:20/1/2026

HOW TO START THIS PROJECT: Abra el poryecto en BlueJ y ejecute el main.

AUTHORS: Kevin Villalobos y Rodrigo Rivera

USER INSTRUCTIONS: Este programa es la primera parte de un proyecto que facilite las elecciones presidenciales en Costa Rica, el proyecto cuenta con 4 clases: Partido, Candidato, Votante y el Main, en las clases Partido, Candidato y Votante se encuentran sus respectivos atributos así como sus Getters y Setters, en el Main se encuentra un menu que esta dentro de un ciclo Do-While que se repetira hasta que la opción elegida sea igual a 5, siendo decision del usuario , funciona de manera que se le da la bienvenida al usuario junto con las instrucciones, una vez se lea la opción elegida entrara en un Switch, por el momento se le informa al usuario que el sistema esta en construccion a esperar de nuevas instrucciones para hacer los metodos correspondientes y los metodos toString de cada clase.
